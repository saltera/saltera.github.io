    Installer missing non-free wifi firmware #

    The Intel 8265/8275 requires non-free firmware whoever this can be added to the installation media.

    Copy the non-free firmware-iwlwifi_*_all.deb from https://cdimage.debian.org/cdimage/unofficial/non-free/firmware/ to firmware folder of the installation media.

    ACPI issue #

    There is currently a bug that means that some kernels (particularly newer ones) will detect the wrong ACPI revision.

    Add acpi_rev_override=5 to /etc/default/grub

    GRUB_CMDLINE_LINUX="acpi_rev_override=5 quiet"

    Update grub

    update-grub
