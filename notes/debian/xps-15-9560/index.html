XPS 15 9560

Dell XPS-9560

Installation issue

Add the non-free firmware-iwlwifi_*_all.deb from https://cdimage.debian.org/cdimage/unofficial/non-free/firmware/ to the installation media /firmware

ACPI issue

There is currently a bug that means that some kernels (particularly newer ones) will detect the wrong ACPI revision.

Add acpi_rev_override=5 to /etc/default/grub

GRUB_CMDLINE_LINUX="acpi_rev_override=5 quiet"

Update grub

update-grub
